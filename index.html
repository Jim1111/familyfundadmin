Accessibility App Package for Family Fund

Apps and resources to cover Autism, Learning Disabilities, Visual Impairment, Physical Impairment and Hearing Impairment

Before Downloading the App Package for iPad please make sure you have configured the following on the iPad:

  • Open the Files app
  • In Locations go to 'On My iPhone'
  • Create a New Folder named 'Family Fund' by pressing a blank area and selecting New Folder
  • Go to Settings\Safari\Downloads and change to On My iPhone
  • Select the Family Fund Folder from the list provided

You will now be able to click the Download button below to install the App Package on the iPad.
